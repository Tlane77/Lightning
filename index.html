JavaScript Object and Variable Exercises

Arrays are collections of similiar things. Objects are ways to define details of a single thing. Both an array and an object can be assigned to a variable.

To obtain the value inside an object, and you know the name of the key, you can use dot notation. Here's the pattern...variableContainingTheObject.keyName

Square Bracket Notation

Square bracket notation is most often used when the key name is stored in a variable. The basic mechanics of it are similar to dot notation, you just use square brackets with the key name inside.// Square bracket notation for key access lassie["breed"] > "Collie"// Dot notation for key access lassie.breed > "Collie"
